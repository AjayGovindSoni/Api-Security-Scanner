API Security Scanner

API Security Scanner is a tool designed to help developers and security professionals assess the security posture of APIs. It provides automated scanning and analysis of REST and HTTP APIs to detect common vulnerabilities and security weaknesses.

This project is based on the original API-Security-Scanner by the elliotsecops GitHub organization, used here under the MIT License with full attribution preserved. The original repository contains documentation, configuration guides, and usage examples that informed this version.

Features

API endpoint scanning for security vulnerabilities

Automated checks against common security issues

Support for configuration-based and rule-driven testing

Reporting of scan results for developer review

Getting Started

Clone the repository

git clone https://github.com/AjayGovindSoni/Api-Security-Scanner.git
cd Api-Security-Scanner


Install dependencies
Follow instructions in the corresponding .md files.

Run the scanner
Refer to available documentation to configure scans and generate reports.
